# How to Setup

1. Pull the code.
2. Create virtual environment using following command: `Linux` - `python3 -m venv .venv`. `Windows` - `python -m venv .venv`.
3. Activate the venv: `Linux` - `source .venv/bin/activate`. `Windows` - `.venv/Scripts/activate`.
4. Install required packages: `pip install -r requirements.txt`.
